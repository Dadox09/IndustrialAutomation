Problem Overview
The deployment of offshore wind farms is a race against time and nature. This assignment
requires you to optimize the sequence of installation tasks for a single Heavy-Lift Vessel
(HLV). You must balance theFuel Consumptioncaused by sequence-dependent setups
(ballasting and positioning) against theCarbon Penaltyof missing weather-dependent
installation windows.
Manufacturing Context: The Offshore Wind Array
The HLV installs three types of structures:Monopiles(M),Jackets(J), andTurbine
Nacelles(N).
•Sequence-Dependent Setups (Fuel Spike):Switching between Monopiles and
Jackets requires a ”Major Ballast Shift” and crane re-rigging. This process con-
sumes a significant amount of diesel fuel.
•Weather Windows (Sustainability Penalty):Each site has a predicted ”Safe
Sea State” window. If the HLV is tardy and misses this window, it must remain on-
station in ”DP-Mode” (Dynamic Positioning), burning auxiliary fuel while waiting
for the next window.
•Environmental Weight (w j):Sites further offshore have higher weights because
the logistics of supporting a delayed vessel there are more carbon-intensive.

The Environmental Penalty Function
Tardiness is penalized based on the cost of ”Dynamic Positioning” (DP) idling:
•IfC j ≤d j: Penalty = 0.
•IfC j > d j: Penalty =w j ×(C j −d j) + Re-mobilization Cost.
TheRe-mobilization Costis a fixed penalty applied if the vessel misses the weather
window entirely, requiring it to retreat and return later.
